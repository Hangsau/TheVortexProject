# Vortex 教學競技框架（物理 gap fuse）

## 核心原理（PubMed + 趨勢 fuse）
- **彈性蓄能**：腿肌腱像彈簧儲能回推（🟢肌腱推進文獻），教學用慢腿蹬建覺，競技高頻回收。
- **自然頻率**：肩踝自振~2-5Hz（🟡陸上振動轉水），匹配划水頻避傷（慢教學，快競技）。
- **渦流推進**：手劃生旋渦拉水前（🟡魚尾間接），肩輕轉重推。
- **fuse 趨勢**：AR 泳鏡看頻（FORM Goggles），AI 校動作（SwimSmooth），VR 練渦（Quest SwimSim）。

## 🔴 gap 轉用（無真人數據→保守+自測）
- 真人泳中肩踝頻無：用2-4Hz 教學安全帶，高5Hz 競技試。
- 渦流肩實測缺：模擬手勢，池測速增。

## 教學動作（初學 5 模板，10min/日）
1. **肩慢圈**：手臂慢圈2Hz，覺彈性拉（AR 鏡頻示警）。
2. **踝彈蹬**：腳尖回彈推，儲能感（VR 看渦形）。
3. **渦手劃**：手掌內旋拉水，輕肩（AI 拍校）。
4. **頻合呼吸**：划3吸1，鎖自然節奏。
5. **綜合游**：50m 慢泳，記舒適頻。

## 競技優化（高階，PR 追蹤）
1. **肩高頻渦**：5Hz 快劃+內旋，推速增10%估。
2. **腿蓄爆發**：深蹬回彈，減拖。
3. **共振鎖定**：頻測匹配，傷防20%。
4. **AR 數據游**：鏡實時頻/渦反饋，迭代 PR。
5. **恢復循環**：高頻後慢蓄，循環賽用。

## 自測指南（手機 App Phyphox/泳錶）
1. **陸測頻**：肩/踝振10s，記基線。
2. **池驗**：3x50m 動作，速/痛/心率記。
3. **反饋**：速升無痛=好，痛停調頻。
4. **工具**：AR 鏡+AI App，GA 平均 3 次。
