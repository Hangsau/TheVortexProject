## 草稿格式規範

每份文件仿 `Research/物理現象/渦流回收.md` 的格式：

1. **理論機制**：物理或生理學原理，用圖示輔助（文字描述流程即可）
2. **文獻**：每條文獻標 DOI 或 PubMed ID，附確定性標記（🟢🟡🔴 等）
3. **🔴 空缺清單**：確認為研究空白的寫「確認為研究空白」，不是只留問號
4. **教學應用**：每個游泳時刻的感知目標語言（物理描述 → 感知目標 → 感知失敗的樣子）

**確定性標記系統：**
```
🔵 邏輯推導    — 物理或生理演繹，不需文獻
🟢 近期文獻    — 有實證支持（需 DOI 可查）
🟡 有效舊文獻  — 物理原理成立，數值可能已更新
🟠 教練觀測    — 實務參考價值，非學術控制組
🔴 個人假設    — 待查，或確認為研究空白
```

---

## 品質紅線（不能越的）

1. **人名歸因是高風險幻覺區**：任何「X 由 Y 創立/發現」，不是從原文直接看到的一律標 🔴 或省略
2. **數字必須追到原始來源**：搜尋摘要的數字不算查證，需要論文或教科書原文
3. **近期文獻（2024+）必須 WebSearch 實際查到才能標 🟢**
4. **確定性層次標記是信任核心**：有疑問先標 🔴，不要為了好看降低門檻

## 彈性蓄能.md 的具體範圍（Task #1）

四個游泳時刻分開分析，每個獨立找文獻或標空白：

| 泳式 | 蓄能時刻 | 涉及構造 |
|------|---------|---------|
| 自由式 | 划手末端 → 入水之間 | 肩帶肌群（背闊肌 / 菱形肌）預拉 |
| 蛙式 | 收腿結束瞬間 | 踝關節 + 髖關節彈性 |
| 仰式 | 旋轉到頂點時 | 背闊肌最適初長度（已在技術分析提及） |
| 蝶式 | 波動傳到踝關節前 | 踝關節彈性 |

文獻起點：跑步阿基里斯腱文獻（Biewener 等）作為對照基準，游泳段大概率是空白，要如實標出。
