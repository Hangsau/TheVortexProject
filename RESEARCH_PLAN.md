# The Vortex Project — 研究規劃架構

> 建立日期：2026-04-10  
> 性質：個人教學研究 × 純粹好奇驅動  
> 核心命題：**所有技術都建立在水感知上**  
> 年限：不設限，依好奇心與教學需求推進

---

## 研究架構總覽

這個計劃分四個研究層，每層有不同的認識論性質：

```
Layer 0 │ 系統完整性      → 把已知的東西做完、做紮實
Layer 1 │ 感知科學基礎    → 理解「水感知」的底層機制
Layer 2 │ 教學觀察研究    → 在真實教學中驗證與發現
Layer 3 │ 物理現象探索    → 探索還沒人問過的問題
        │
        ↓
整合層  │ 持續把發現寫回系統
```

四層不是線性推進，是**同時存在的研究場域**，每次學習或教學都可能在不同層產生進展。

---

## Layer 0 — 系統完整性

**性質：補完已知，消除邏輯缺口**  
**方法：寫作與整理**  
**問題：這套系統在邏輯上是否閉環？**

這一層不是研究，是研究的前提。系統有邏輯缺口，後面的觀察和探索就會在錯誤的地基上進行。

### 0.1 自由式水感框架（缺失）

Technica/ 目前有仰式 v4.2、蛙式 v5、蝶式 v2，唯獨自由式缺分式水感框架。  
自由式是多數學生接觸最多的泳式，這個缺口影響最廣。

**待完成：**
- 自由式 L0–L6 水感框架
- 三型診斷在自由式的具體表現（A型：EVF崩潰；B型：踢水節律失守；C型：換氣觸發感知遮蔽）
- 感知缺陷 vs 硬體邊界的前置判斷（前鋸肌耐力是硬體問題，不是感知問題）

### 0.2 感知目標語言（四式技術分析的缺失層）

技術分析描述「動作的物理條件」，但沒有描述「泳者應該感覺到什麼」。  
這是 Instructional/ 和 Technica/ 之間缺失的橋。

**格式：** 每個核心技術點補一條感知目標

```
技術點：EVF（高肘捕水）
物理描述：上臂平行游進方向，手掌前臂面向後方
感知目標：感覺手掌和前臂同時「頂到」阻力，
         而不是手先滑過去、前臂才接到
感知失敗的樣子：感覺只有手掌在抓水，前臂是空的
```

**四式各需補充的核心技術點：**

| 泳式 | 關鍵技術點（需補感知目標） |
|------|--------------------------|
| 自由式 | EVF、旋轉連動、前鋸肌疲勞感知、弓形波換氣 |
| 仰式 | 肩旋轉不對稱性、小指入水、踢水對位 |
| 蛙式 | insweep 加速、渦流蹬入感、收腿路徑 |
| 蝶式 | 波動觸發（頭+胸下壓）、第一踢vs第二踢差異、Recovery停頓 |

### 0.3 感知層次 3–5 的訓練方法（目前空白）

水感完整指南建立了五個感知層次的概念，但 L3 以上的訓練方法極度薄弱：

| 感知層次 | 感知對象 | 訓練方法現況 |
|---------|---------|------------|
| 1 壓力方向 | 手掌受力方向 | 有（拳頭游、Sculling） |
| 2 流速差異 | 水移動快慢 | 有（速度對比訓練） |
| 3 渦流性質 | 阻力性質改變 | **缺失** |
| 4 時序交接 | 動作間能量移交 | **缺失** |
| 5 全身場感知 | 整體水場狀態 | **缺失** |

**待完成：** 為層次 3–5 各設計至少一種可操作的訓練方法

### 0.4 出發、轉身、水下海豚踢

三份技術分析（自由式、蛙式、蝶式）幾乎沒有涵蓋出發與轉身。  
仰式文件有提到，但也不完整。

González-Ravé（2025）確認：100m 仰式中出發+轉身佔整場約 1/3 時間。  
其他泳式比例相近。這不是邊緣細節，是被系統性忽略的核心技術區塊。

## Layer 1 — 感知科學基礎

**性質：理解「水感知」的底層機制**  
**方法：文獻閱讀與理論建構**  
**問題：水感知是什麼？它如何建立？它的上限在哪裡？**

這一層的目標不是「做研究」，而是把神經科學、感知心理學、運動學習的既有知識，系統地整合進這個框架。

### 1.1 機械感受器地圖

人體皮膚有四種主要機械感受器，各有不同的感知特性：

| 感受器 | 感知特性 | 游泳中的可能角色 |
|--------|---------|----------------|
| Meissner 小體 | 低頻振動、輕觸 | 手掌感覺水流滑過 |
| Merkel 細胞 | 持續壓力、精細空間解析 | 手掌感覺水壓方向 |
| Ruffini 末梢 | 皮膚拉伸、深壓 | 感覺水流對皮膚的切應力 |
| Pacinian 小體 | 高頻振動 | 感覺渦流產生的微振動？ |

**研究問題：**
- 每種感受器的密度在哪些身體部位最高？（指尖 > 前臂 > 大腿——但精確地圖？）
- 長期游泳訓練者與非游泳者，手部觸覺辨別能力有差異嗎？
- 感知剝奪訓練（拳頭游）的神經機制是「去適應後重適應」還是別的？

**文獻起點：** Johansson & Flanagan（2009）手部感覺運動控制；Tremblay et al. 觸覺辨別訓練研究

### 1.2 Feedforward 固化機制

技術分析中提到了 feedforward，但沒有深入展開。這是感知科學最核心的概念之一：

```
Feedback 控制（初學）：
感知到阻力 → 大腦處理 → 送出調整指令 → 動作改變
延遲：100–200ms，在比賽速度下來不及

Feedforward 控制（熟練）：
大腦預測接下來會發生什麼 → 提前送出指令
延遲：幾乎為零，但需要大量正確重複建立預測模型
```

**研究問題：**
- Feedforward 固化需要多少次正確重複？（運動學習文獻有估計值嗎？）
- 慢速練習 vs 比賽速度練習，哪個更有效建立 feedforward？
- 水感剝奪訓練如何加速 feedforward 的固化？

**文獻起點：** Wolpert & Flanagan（2001）內部模型理論；Schmidt & Lee 運動學習教科書

### 1.3 感知學習（Perceptual Learning）

感知學習是一個獨立的研究領域，研究感知精度如何透過訓練提升。

核心發現（Eleanor Gibson 以降）：
- 感知學習是真實的神經可塑性，不只是「注意力」的改變
- 差異化訓練（讓學習者辨別細微差異）比重複訓練更有效
- 感知學習往往是自動化的，不需要明確意識介入

**這對水感訓練的含義：**
- 不是「游更多」就能建立更好的水感
- 需要刻意製造感知差異讓神經系統辨別
- 感知剝奪訓練正是差異化訓練的一種形式

**研究問題：**
- 感知學習的速度是否有個體差異的生物基礎？
- 水感的感知學習是否有「關鍵期」（敏感期）？

**文獻起點：** Gibson（1969）感知學習理論；Fine & Jacobs（2002）感知學習綜述

### 1.4 隱性學習 vs 顯性學習

這直接關係到「為什麼不直接教動作形式」的理論基礎：

**顯性學習**（直接教規則）：快，但脆弱——壓力下容易崩潰  
**隱性學習**（在環境中探索）：慢，但穩健——壓力下維持

Masters（1992）的研究：高壓力下，顯性學習者的技術崩潰速度遠快於隱性學習者。

**研究問題：**
- L0–L6 框架的哪些階段主要走隱性學習路徑？哪些需要顯性？
- 感知目標語言是在促進顯性還是隱性學習？
- 是否有辦法讓顯性理解加速隱性學習？（而不是取代它）

**文獻起點：** Masters（1992）；Liao & Masters（2001）；Maxwell et al. 類比學習研究

## Layer 2 — 教學觀察研究

**性質：在真實教學中觀察、記錄、找模式**  
**方法：系統化案例記錄**  
**問題：這套框架在真實教學中的行為是什麼？**

這一層不需要實驗設計，需要的是**系統化的觀察習慣**。

### 2.1 建立觀察記錄系統

每個學生建立一份觀察檔案，記錄：

```
基本資料
├── 游泳年資、教學背景
├── 入學時的感知層次評估（L 級）
├── 三型診斷結果（A/B/C）
└── 解剖邊界評估（踝關節、髖內旋、肩活動度）

訓練記錄
├── 每次訓練的主要訓練方法
├── 學生的感知描述語言（他們怎麼描述自己的感覺）
├── 技術變化的外部觀察指標
└── 瓶頸點：卡在哪裡、卡多久

進展追蹤
├── L 級提升的觸發事件（是什麼讓他們突破的？）
├── 三型診斷的變化（有沒有從一型轉到另一型？）
└── 技術在壓力/疲勞下的穩定性
```

### 2.2 核心觀察問題

這些是有答案後，整個框架會變得更紮實的問題：

**關於三型診斷：**
- A型（手感缺失）、B型（腳感缺失）、C型（全身張力問題）——這個分類在教學現場的預測力有多準？
- 一個學生被診斷為 A型後，使用 A型訓練方法，進展速度比沒有分型快嗎？
- 有沒有第四型的存在？（目前框架是否有未被涵蓋的感知缺陷模式？）

**關於感知剝奪訓練：**
- 「復活感」（脫掉干擾工具後的高感知窗口）平均持續多久？
- 在哪個 L 級，感知剝奪訓練效果最顯著？
- 拳頭游 vs 穿襪踢水 vs 其他剝奪工具，效果有可比較的差異嗎？

**關於感知與技術穩定性：**
- 感知優先訓練的學生，在疲勞下的技術崩潰序列，跟形式優先訓練的學生有什麼不同？
- 一個技術動作「正確但感知未建立」和「感知已建立但形式還不穩定」，哪個在壓力下更穩健？

**關於個體差異：**
- 感知學習速度最快的學生，有什麼共同特徵？（運動背景？語言描述能力？）
- 解剖邊界和感知發展速度有相關嗎？（踝關節限制 → 蛙腿感知建立困難？）

### 2.3 觀察結果的記錄格式

累積到一定數量後，可以開始找模式。建議的記錄格式：

每個「突破事件」記錄：
```
泳者：L級、三型診斷
訓練方法：做了什麼
感知描述：學生說感覺到了什麼
外部觀察：動作上出現什麼變化
觸發因素：是什麼讓這次突破發生的？
```

這些案例是未來最有價值的研究素材。

## Layer 3 — 物理現象探索

**性質：對未知領域提出有根據的假設**  
**方法：文獻深挖 + 實際水中探索**  
**問題：游泳中有哪些物理現象，目前的技術理解還沒有觸碰到？**

這一層的每個方向都需要做一次「文獻清算」：現有文獻說了什麼、沒說什麼、下一步問題是什麼。

### 3.1 渦流回收（Vortex Recycling）— 優先探索

**現有文獻基礎：**
- Tanaka et al.（2024/2025）蛙式踢腿 CFD——最直接的實證
- Triantafyllou & Triantafyllou（1995）魚類渦流推進
- Liao et al.（2003）魚在卡門渦街中的被動游進

**文獻清算任務：**
Tanaka 2024 的 CFD 數據具體顯示了什麼？渦流在踢腿結束後多久消散？速度？空間範圍？這些細節決定了「接收窗口」有多窄。

**可以在水中探索的：**
蛙式踢腿後，下一個外划的啟動時機對推進力的影響——用 SWOLF 作為代理指標，系統性測試不同滑行時機。

**假設（待驗證）：**
存在一個「最優啟動時機」，在這個時機啟動外划的 SWOLF 顯著低於提前或延遲啟動。這個時機可能就是渦流回收窗口。

**感知連結：**
如果渦流回收窗口存在，可以教學生感知的訓練方法是什麼？「感覺到水從後面輕推腳」可能是渦流接收的感知描述。

---

### 3.2 身體自然頻率（Resonant Frequency）

**現有文獻基礎：**
- Toussaint & Beek（1992）游泳的機械效率測量
- Fernandes et al. 不同划頻下的氧耗研究
- 仿生游泳（bionic swimming）的頻率-效率關係

**文獻清算任務：**
有沒有研究直接測量同一泳者在不同划頻下的氧耗-速度曲線？效率峰值是否存在、且是個體化的？

**核心假設：**
每個泳者存在一個「效率峰值划頻」（最低氧耗/速度比），這個频率不等於最高速度划頻，且是個體化的。

**可以在教學中探索的：**
讓學生在不同划頻下游相同距離，記錄：
1. 主觀努力感（RPE）
2. 速度
3. 技術穩定性（SPL 波動）

觀察是否有某個划頻出現「明顯省力但速度不下降」的窗口。

---

### 3.3 彈性蓄能（Elastic Rebound）

**現有文獻基礎：**
- 跑步中的阿基里斯腱蓄能：Ker et al.（1987）
- 游泳中的腱彈性：研究極少，是真正的空白
- 仰式技術分析已提到「背闊肌最適初長度」——這是彈性蓄能的近似概念

**文獻清算任務：**
有沒有任何游泳研究測量過肌腱的彈性能量儲存？EMG 靜默期（肌肉不主動收縮）是否存在於划手週期中？

**感知連結：**
彈性蓄能的感知目標可能是：「感覺到動作在末端被『彈』出去，而不是被『推』出去。」

這個感知差異如果存在且可訓練，可能是 L5→L6 過渡的關鍵感知之一。

---

### 3.4 皮膚機械感受器訓練邊界

**這是與整個系統關係最直接的前沿方向。**

**現有文獻基礎：**
- Tremblay et al.（2003）觸覺辨別訓練可以提升感知精度
- Godde et al.（2000）被動觸覺刺激引發皮質重組
- Johansson & Flanagan（2009）手部感覺運動整合

**文獻清算任務：**
感知剝奪→恢復的神經機制，有沒有研究直接量化「敏化窗口」的時長和強度？

**核心假設：**
感知剝奪訓練的效果，不只是「讓人注意到感覺」（心理層次），而是短暫提升機械感受器的信噪比（神經層次）。兩種機制對教學設計的含義不同。

**可以在教學中觀察的：**
拳頭游結束後，學生能維持「復活感」的時長——這個時長是否可以透過訓練延長？

## 整合層 — 研究如何回饋系統

每個層次的發現，都需要找到它在系統中的位置：

```
Layer 0 的完成
→ 直接更新 Instructional/ 和 Technica/ 文件

Layer 1 的文獻整合
→ 為水感完整指南增加「神經科學基礎」章節
→ 為 L0–L6 框架提供更精確的機制解釋

Layer 2 的教學觀察
→ 更新三型診斷的準確性和適用範圍
→ 為感知剝奪訓練建立「最有效的使用時機」指引
→ 累積感知目標語言的真實案例（學生怎麼描述突破瞬間）

Layer 3 的物理探索
→ 以「教練可操作的形式」寫回對應技術分析
→ 不成熟的假設放在 FUTURE_RESEARCH.md，
  成熟的發現寫入 Instructional/
```

## 研究的態度原則

**區分確定性層次**（這套系統已在做，要繼續保持）

```
🔵 邏輯推導    — 物理或生理學演繹，原則上跨時間有效
🟢 近期文獻    — 有實證支持，但有信賴區間
🟡 有效舊文獻  — 物理原理仍成立，具體數值可能已更新
🟠 教練觀測    — 有實務參考價值，非學術控制組
🔴 個人假設    — 目前是直覺，需要更多文獻或觀察支持
```

**研究不是為了得到「正確答案」**

每個層次的研究，都可能讓你發現：
- 現有框架的某個假設是錯的
- 某個現象比想像的複雜
- 某個被認為重要的東西，實際上沒那麼重要

這些發現和「確認假設」同樣有價值。
