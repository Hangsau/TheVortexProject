## 一、水下蝶腿 vs 水面蝶踢——差異比對

> 主文件目前完全沒有這個比對。這是教學現場最常被混淆的地方。

---

### 1.1 上踢 vs 下踢的推進貢獻比例

**缺什麼**：水面蝶式下踢是主推進踢，水下蝶腿則是上踢速度峰值才是關鍵指標——這個翻轉在主文件 3.2 節只提到結論，沒有說明「為什麼水下和水面不一樣」。

**查詢方向**：
```
underwater dolphin kick upkick propulsion biomechanics
dolphin kick upkick downkick propulsive contribution
butterfly kick vs underwater dolphin kick kinematic difference
upkick peak toe velocity swimming
```

**可能來源**：
- MDPI（Sports, Biomechanics 期刊）2018–2024
- Frontiers in Physiology / Frontiers in Sports
- Journal of Sports Sciences
- 研究群體：Tor, Pease, Ball（澳洲研究群）；Hochstein & Blickhan（德國）

**填補後標注**：🟢（若找到直接研究）/ 🔵（若從力學推導）

**優先級**：★★★

---

### 1.2 振幅邏輯：水面需要大振幅，水下要小振幅

**缺什麼**：為什麼水面蝶踢需要足夠振幅（讓身體浮出水面），但水下反而是小振幅高頻最優——截面積增加 vs 推進力增加的量化分析。

**查詢方向**：
```
underwater dolphin kick amplitude frequency optimal swimming
dolphin kick amplitude drag trade-off
butterfly kick amplitude hydrodynamics wave drag
frontal area drag swimming undulation amplitude
```

**可能來源**：
- Journal of Biomechanics
- Human Movement Science
- Lyttle 系列研究（1998–2000）的延伸引用

**填補後標注**：🔵（物理推導）+ 🟢（若有量化研究）

**優先級**：★★★

---

### 1.3 驅動方向與身體位置的差異

**缺什麼**：水面蝶踢要對抗重力製造向上推升力；水下流線型姿勢讓力量主要轉為水平推進——兩者的力學目標根本不同，但沒有文件直接對比。

**查詢方向**：
```
surface butterfly kick lift force vertical propulsion
underwater dolphin kick horizontal propulsion streamline position
dolphin kick body position hydrodynamic force direction
```

**可能來源**：
- 運動生物力學教科書（Hay, Counsilman）
- Journal of Applied Biomechanics

**填補後標注**：🔵（推導為主）

**優先級**：★★

## 二、水下蝶腿的身體協調機制——腰、胸、腿怎麼運作

## 三、爭議歷史——曾有爭議、現在有結論的案例

> 這類內容最有教學與知識史價值，應寫成「爭議已解決」的完整段落，附時間軸。

---

### 3.1 【已解決】「上踢只是恢復動作，無推進功能」

**爭議期**：1990年代–2010年代初期主流教學觀點

**舊觀點**：下踢產生推進力，上踢只是讓腿部回位的恢復動作，不貢獻前進速度。

**翻轉時間**：2010年代中後期，隨 force plate、3D kinematic 技術進步

**現有結論**：上踢的腳尖峰值速度（peak vertical toe velocity during upkick）是 UDK 效率的關鍵預測因子（MDPI 2022；Frontiers 2021）。上踢並非無效的恢復動作。

**查詢方向**：
```
dolphin kick upkick propulsion history review
underwater kick upstroke downstroke contribution swimming
"upkick" "toe velocity" dolphin kick elite
dolphin kick force production upward downward phase
```

**可能來源**：
- MDPI Sports / Biomechanics 2020–2024 系統回顧
- Hochstein & Blickhan 2011（早期翻轉的關鍵論文之一）
- Nakashima 系列（水動力模擬研究）

**填補後標注**：🟢 + 爭議史說明

**優先級**：★★★

---

### 3.2 【已解決】「起踢越早越好」

**爭議期**：2000年代–2010年代

**舊觀點**：入水後立刻開始踢腿，最大化水下踢腿時間。

**現有結論**：先滑行至速度閾值後再起踢才是最優（Sports Biomechanics, 2026）。過早起踢干擾流線型，且在初始速度遠高於踢腿速度時踢腿反而是負貢獻。

**查詢方向**：
```
underwater dolphin kick initiation timing optimal glide
start glide duration dolphin kick transition speed threshold
streamline to kick timing swimming start
optimal UDK start timing biomechanics
```

**可能來源**：
- Sports Biomechanics 2026（主文件已引用，需確認全文）
- Vantorre et al. 系列（出發水下階段分析）

**填補後標注**：🟢

**優先級**：★★

---

### 3.3 【仍有爭議】仰式水下是否應全程仰臥踢腿

**爭議核心**：仰臥姿的 UDK 是否和俯臥姿等效？部分研究與教練認為側轉至半俯臥（side-lying）能讓踢腿更有力。

**查詢方向**：
```
backstroke underwater dolphin kick body position
supine vs prone dolphin kick efficiency
backstroke UDK rotation body roll effectiveness
backstroke start underwater kinematic elite
```

**可能來源**：
- Frontiers in Sports（有仰式水下分析的相關研究）
- International Journal of Sports Physiology and Performance

**填補後標注**：🔴（仍待查）或 🟠

**優先級**：★★

---

### 3.4 【仍有爭議】「訓練期應先練距離再練速度」vs「從一開始只練速度」

**爭議核心**：教練界的訓練哲學分歧——讓學員先以距離為目標建立習慣，之後再精緻化；vs 從一開始就以速度和效率為目標，即使距離短也沒關係。

**查詢方向**：
```
dolphin kick training distance vs speed focus coaching
UDK training approach youth swimmers
underwater dolphin kick skill acquisition motor learning
```

**可能來源**：
- 可能沒有直接研究，以教練訪談和實務觀測為主
- Motor learning 的一般文獻可以提供框架

**填補後標注**：🟠（教練觀測）

**優先級**：★
